# Student Performance Dashboard

This project is an interactive Excel dashboard created to analyze student performance data and identify academic trends.

## Features
- Subject-wise Performance Analysis
- Student Score Comparison
- Attendance Insights
- Performance Trends
- Interactive Filters and Charts

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
