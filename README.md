# 📚 Overview

The **Student Performance Dashboard** is an interactive data analytics project built using Microsoft Excel to analyze student academic performance and identify meaningful educational trends.

This dashboard provides visual insights into:
- 📈 Academic performance
- 📊 Subject-wise score analysis
- 🧑‍🎓 Student comparisons
- 📅 Attendance tracking
- 📉 Performance trends over time

The project focuses on transforming raw educational data into clear and actionable visual insights.

---

# ✨ Features

## 📖 Subject-wise Performance Analysis
- Analyze marks across different subjects
- Identify strongest and weakest subjects
- Compare subject performance visually

---

## 🧑‍🎓 Student Score Comparison
- Compare student scores interactively
- Highlight high and low performers
- Evaluate overall academic consistency

---

## 📅 Attendance Insights
- Attendance tracking and visualization
- Correlation between attendance and scores
- Identify attendance-related performance trends

---

## 📈 Performance Trends
- Observe academic growth patterns
- Analyze score fluctuations over time
- Detect performance improvement areas

---

## 🎛️ Interactive Dashboard Elements
- Dynamic charts 📊

- Filter-based analysis ⚡
- User-friendly layout 🖥️
